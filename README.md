🤔 남아 있는 공간을 빌려주고 수익을 창출할 수 있지 않을까?<br>
🥺 친구들과 추억을 쌓을 프라이빗한 공간 어디 없을까?<br>

<br>

🎤소개
--
'여기다!'는 공간 예약 플랫폼 입니다.<br>
운영자는 사용되지 않는 공간을 개성있게 꾸며 이용자들에게 제공함으로써 수익을 발생시킬 수 있고,<br>
사용자는 특별한 이벤트나 사적인 공간으로 활용하여 추억을 쌓을 수 있도록 도와줍니다.<br>
